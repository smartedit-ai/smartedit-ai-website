# 智能助手

SmartEdit AI 的右侧智能助手使用 Chrome 原生 Side Panel API，提供独立的浏览器侧边面板。

## 🎯 功能概览

右侧智能助手包含四大模块：

- **快捷工具** - 字数统计、链接复制、二维码生成等
- **快速笔记** - 随时记录想法，自动保存
- **AI 助手** - 文本总结、翻译、润色、扩写
- **页面信息** - 查看页面元数据和统计信息

---

## 🚪 打开智能助手

### 方式一：通过扩展图标

1. 点击浏览器工具栏的 SmartEdit AI 图标
2. 点击「右侧助手」按钮
3. 右侧会打开独立的侧边面板

### 方式二：通过右键菜单

1. 在任意网页右键
2. 选择「智编助手」→「打开右侧助手」
3. 侧边面板在右侧打开

### 特点

**与传统浮层的区别：**
- ✅ 独立的浏览器面板，不是浮动遮罩
- ✅ 使用浏览器原生 API，性能更好
- ✅ 浏览器自动调整页面宽度
- ✅ 切换标签页时保持打开状态
- ✅ 按 ESC 键快速关闭

## 🔧 快捷工具

### 字数统计

**功能：** 统计当前页面或选中文字的字数

**使用方法：**
1. 打开右侧助手
2. 切换到「快捷工具」标签
3. 点击「字数统计」

**统计信息：**
- 总字数
- 中文字符数
- 英文单词数
- 段落数
- 预计阅读时间

**使用场景：**
- 检查文章长度
- 控制字数要求
- 评估阅读时间

### 复制链接

**功能：** 快速复制当前页面的链接和标题

**提供选项：**
- 复制 URL
- 复制标题
- 复制 Markdown 格式链接
- 复制 HTML 格式链接

**Markdown 格式：**
```markdown
[页面标题](https://example.com)
```

**HTML 格式：**
```html
<a href="https://example.com">页面标题</a>
```

**使用场景：**
- 分享文章链接
- 引用参考资料
- 制作链接列表

### 生成二维码

**功能：** 为当前页面生成二维码

**使用方法：**
1. 点击「生成二维码」
2. 自动生成当前页面的二维码
3. 可以下载或分享

**使用场景：**
- 移动端访问
- 分享给他人
- 打印传播

### 页面翻译

**功能：** 快速翻译当前页面

**支持的翻译：**
- 中文 → 英文
- 英文 → 中文
- 其他语言互译

**使用方法：**
1. 点击「页面翻译」
2. 选择目标语言
3. 查看翻译结果

**使用场景：**
- 阅读外文资料
- 翻译文章内容
- 学习外语

## 📝 快速笔记

### 功能说明

随时记录想法和灵感，自动保存到浏览器本地存储。

### 使用方法

**1. 打开笔记**
- 切换到「快速笔记」标签
- 在文本框中输入内容

**2. 自动保存**
- 输入内容会自动保存
- 无需手动点击保存按钮
- 关闭浏览器后内容不会丢失

**3. 导出笔记**
- 点击「导出」按钮
- 选择导出格式：
  - 纯文本 (.txt)
  - Markdown (.md)
  - HTML (.html)

**4. 清空笔记**
- 点击「清空」按钮
- 确认后清空所有内容

### 功能特点

**实时保存：**
- 每次输入都会自动保存
- 不用担心内容丢失

**字符计数：**
- 实时显示字符数
- 了解笔记长度

**Markdown 支持：**
- 支持 Markdown 语法
- 导出时保留格式

**多设备同步：**

云端同步功能正在开发中。

**灵感记录：**

**临时笔记：**
- 会议要点

## ✨ AI 助手

### 功能说明

快速处理文本内容，支持总结、翻译、润色、扩写等操作。

### 文本总结

**功能：** 将长文本总结为简短摘要

**使用方法：**
1. 在文本框输入或粘贴内容
2. 或选中页面文字后点击「使用选中文字」
3. 点击「总结」按钮
4. 查看 AI 生成的摘要

**总结类型：**
- **简短摘要** - 1-2 句话概括
- **要点提取** - 列出关键要点
- **详细总结** - 段落式总结

**使用场景：**
- 快速了解文章内容
- 提取核心观点
- 制作内容摘要

**示例：**

**原文（500字）：**
> "人工智能技术在近年来取得了突破性进展...（省略）"

**总结（50字）：**
> "人工智能技术快速发展，在医疗、教育、金融等领域广泛应用，但也面临数据隐私、算法偏见等挑战，需要建立完善的监管机制。"

### 中英互译

**功能：** 中文和英文之间的互译

**使用方法：**
1. 输入要翻译的文本
2. 点击「翻译」按钮
3. AI 自动识别语言并翻译

**翻译特点：**
- 智能识别语言
- 保持原文语气
- 符合表达习惯
- 准确度高

**使用场景：**
- 翻译外文资料
- 撰写英文内容
- 学习外语

**示例：**

**中译英：**
> 原文："人工智能正在改变我们的生活方式。"
> 译文："Artificial intelligence is transforming the way we live."

**英译中：**
> 原文："The future of AI is promising."
> 译文："人工智能的未来充满希望。"

### 内容润色

**功能：** 优化文字表达，提升文章质量

**使用方法：**
1. 输入要润色的文本
2. 点击「润色」按钮
3. 查看优化后的内容

**润色方向：**
- 优化用词
- 改善句式
- 增强表达力
- 提升文采

**使用场景：**
- 优化文章表达
- 提升文字质量
- 改善写作风格

**示例：**

**原文：**
> "这个方法很好，大家可以试试。"

**润色后：**
> "这个方法经过实践验证，效果显著，强烈推荐大家尝试应用。"

### 文本扩写

**功能：** 扩充内容，增加细节

**使用方法：**
1. 输入简短的文本
2. 点击「扩写」按钮
3. 获得扩充后的内容

**扩写策略：**
- 增加细节描述
- 补充背景信息
- 添加例子说明
- 丰富表达方式

**使用场景：**
- 内容过于简短
- 需要增加细节
- 丰富文章内容

**示例：**

**原文：**
> "时间管理很重要。"

**扩写后：**
> "时间管理对于现代职场人士来说至关重要。良好的时间管理不仅能够显著提升工作效率，还能帮助我们在繁忙的工作中保持清晰的头脑，实现工作与生活的平衡。通过合理安排时间，我们可以更好地完成任务，减少焦虑，提高生活质量。掌握时间管理技巧，就等于掌握了人生的主动权。"

### 使用选中文字

**功能：** 直接使用页面选中的文字

**使用方法：**
1. 在页面上选中文字
2. 打开右侧助手
3. 点击「使用选中文字」
4. 选中的文字自动填入文本框
5. 选择要执行的操作

**优势：**
- 无需手动复制粘贴
- 快速处理页面内容
- 提升操作效率

## ❓ 常见问题

### Q: 侧边栏无法打开？

**可能原因：**
1. 浏览器版本过低
2. 扩展未正确加载
3. 权限不足

**解决方法：**
1. 确认 Chrome 版本 114+
2. 重新加载扩展
3. 刷新页面重试

### Q: 笔记内容丢失？

**原因：**
- 清除了浏览器数据
- 卸载了扩展

**预防方法：**
- 定期导出笔记
- 重要内容及时备份
- 使用云端同步（即将支持）

### Q: AI 功能无响应？

**检查：**
1. 是否配置了 API Key
2. 网络连接是否正常
3. API 额度是否充足

**解决：**
- 查看 [安装配置](/guide/installation)
- 测试 API 连接
- 更换服务商

### Q: 页面信息不准确？

**原因：**
- 页面未完全加载
- 动态内容未渲染
- 页面结构特殊

**解决：**
- 等待页面完全加载
- 刷新后重试
- 手动验证信息
