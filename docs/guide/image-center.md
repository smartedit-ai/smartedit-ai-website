# 配图中心

SmartEdit AI 集成了 Unsplash 和 Pixabay 两大免费图片平台，提供海量高质量图片资源。

## 🖼️ 功能概览

配图中心提供以下功能：

- **图片搜索** - 搜索 Unsplash 和 Pixabay 的海量图片
- **一键插入** - 点击图片直接插入编辑器
- **背景模板** - 预设渐变和纯色背景
- **图片预览** - 大图预览和详细信息
- **免费商用** - 所有图片均可免费商用

## 🚀 使用方法

### 打开配图中心

**方式一：通过侧边栏**
1. 点击扩展图标
2. 选择「左侧侧边栏」
3. 切换到「配图中心」标签

**方式二：通过右键菜单**
1. 在页面右键
2. 选择「智编助手」→「配图中心」

### 搜索图片

**1. 输入关键词**
- 在搜索框输入关键词
- 支持中文和英文
- 可以使用多个关键词

**2. 选择平台**
- Unsplash - 高质量摄影
- Pixabay - 多样化素材
- 或同时搜索两个平台

**3. 浏览结果**
- 瀑布流展示搜索结果
- 滚动加载更多图片
- 每页显示 20-30 张图片

**4. 查看详情**
- 鼠标悬停查看图片信息
- 点击图片查看大图
- 查看作者、尺寸等信息

### 插入图片

**方法一：直接插入**
1. 在编辑器中定位光标
2. 点击图片
3. 图片自动插入

**方法二：预览后插入**
1. 点击图片查看大图
2. 确认满意后点击「插入」
3. 图片插入到编辑器

**方法三：下载后使用**
1. 点击「下载」按钮
2. 保存到本地
3. 手动上传使用

### 调整图片

插入后可以：
- 调整图片大小
- 添加图片说明
- 设置对齐方式
- 添加链接

## ⚙️ 配置图片服务

### 配置 Unsplash

**1. 获取 Access Key**

访问 [Unsplash Developers](https://unsplash.com/developers)：
1. 注册并登录账号
2. 点击「New Application」
3. 填写应用信息：
   - Application name: SmartEdit AI
   - Description: 个人使用
4. 同意服务条款
5. 创建应用
6. 复制 Access Key

**2. 配置到扩展**

1. 打开设置页面
2. 切换到「图片服务」标签
3. 在 Unsplash 部分粘贴 Access Key
4. 点击「测试连接」
5. 显示成功即可使用

**使用限制：**
- 免费版：50 次请求/小时
- 足够个人使用
- 如需更多可申请升级

### 配置 Pixabay

**1. 获取 API Key**

访问 [Pixabay API](https://pixabay.com/api/docs/)：
1. 注册账号
2. 登录后进入「API」页面
3. 在个人设置中找到 API Key
4. 复制 API Key

**2. 配置到扩展**

1. 打开设置页面
2. 在 Pixabay 部分粘贴 API Key
3. 点击「测试连接」
4. 显示成功即可使用

**使用限制：**
- 免费版：5000 次请求/小时
- 完全够用
- 无需升级

### 不配置也能用

**即使不配置 API Key，你仍然可以：**
- ✅ 使用预设的背景模板
- ✅ 使用渐变背景
- ✅ 使用纯色背景
- ❌ 无法搜索 Unsplash 图片
- ❌ 无法搜索 Pixabay 图片

## 💡 搜索技巧

### 关键词选择

**使用具体的关键词：**
```
✅ 好的关键词：
- "商务会议"
- "咖啡店工作"
- "日落海滩"
- "科技感背景"

❌ 不好的关键词：
- "图片"
- "好看"
- "漂亮"
```

**使用英文关键词：**
- Unsplash 主要是英文内容
- 英文关键词搜索结果更丰富
- 可以使用翻译工具

**组合关键词：**
```
单个关键词：coffee
组合关键词：coffee shop laptop work

效果更精准！
```

### 搜索策略

**1. 从宽泛到具体**
```
第一次：business
第二次：business meeting
第三次：business meeting office
```

**2. 尝试同义词**
```
如果 "happy" 没找到满意的
试试：joyful, cheerful, delighted
```

**3. 使用场景描述**
```
不要只搜：computer
试试：person using computer in office
```

**4. 指定风格**
```
添加风格词：
- minimal（极简）
- vintage（复古）
- modern（现代）
- colorful（多彩）
```

### 图片选择

**考虑文章主题：**
- 文章讲什么，图片就配什么
- 避免文不对题

**注意图片质量：**
- 选择高分辨率图片
- 避免模糊、失真
- 注意构图和色彩

**保持风格统一：**
- 全文图片风格一致
- 色调协调
- 尺寸比例统一

**避免版权问题：**
- 使用 Unsplash/Pixabay 的图片
- 都是免费商用
- 无需担心版权

## ❓ 常见问题

### Q: 搜索没有结果？

**可能原因：**
1. 关键词太具体或太生僻
2. 网络连接问题
3. API Key 配置错误
4. 达到请求限制

**解决方法：**
1. 更换关键词，使用更通用的词
2. 检查网络连接
3. 重新配置 API Key
4. 等待一段时间后再试

### Q: 图片无法插入？

**可能原因：**
1. 网络问题导致图片加载失败
2. 编辑器权限限制
3. 图片格式不支持

**解决方法：**
1. 检查网络连接
2. 刷新页面重试
3. 尝试下载后手动上传

### Q: 图片质量不好？

**解决方法：**
1. 使用 Unsplash 搜索高质量图片
2. 选择分辨率更高的图片
3. 避免过度放大图片

### Q: 如何找到合适的图片？

**建议：**
1. 明确图片用途
2. 使用精准的关键词
3. 多尝试不同搜索词
4. 浏览多页结果
5. 参考优秀文章的配图

### Q: 达到请求限制怎么办？

**Unsplash（50次/小时）：**
1. 等待一小时后恢复
2. 申请升级套餐
3. 临时使用 Pixabay

**Pixabay（5000次/小时）：**
- 基本不会达到限制
- 如果达到，等待一小时
