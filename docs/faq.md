# 常见问题 FAQ

本页面收集了用户最常遇到的问题和解决方案。

## 📦 安装相关

### Q: 支持哪些浏览器？

**A:** SmartEdit AI 目前支持：
- ✅ Chrome 114+
- ✅ Edge 114+
- ✅ 其他 Chromium 内核浏览器（Brave、Opera 等）
- ❌ Firefox（暂不支持，因为不支持 Side Panel API）
- ❌ Safari（暂不支持）

### Q: 如何安装扩展？

**A:** 有两种安装方式：
1. **Chrome 应用商店**（即将上线）- 一键安装，自动更新
2. **开发者模式** - 从源码构建，适合开发者

详细步骤请查看 [快速开始](/getting-started) 或 [安装配置](/guide/installation)。

### Q: 安装后图标不显示怎么办？

**A:** 尝试以下方法：
1. 刷新扩展页面（`chrome://extensions/`）
2. 重新加载扩展
3. 运行图标生成脚本：
```bash
npm install sharp
node scripts/generate-icons.js
npm run build
```

### Q: 提示"无法加载扩展程序"？

**A:** 请检查：
1. Chrome 版本是否为 114+
2. 是否已开启「开发者模式」
3. 是否选择了正确的 `dist` 文件夹
4. 尝试重新构建：`npm run build`

## ⚙️ 配置相关

### Q: 必须配置 API Key 吗？

**A:** 是的，AI 功能需要配置 API Key。但样式库、配图中心等功能可以在不配置的情况下使用。

### Q: 哪个 AI 服务商最推荐？

**A:** 根据不同需求推荐：
- **性价比最高**：DeepSeek（¥1/1M tokens）
- **质量最好**：OpenAI GPT-4o
- **国内最快**：智谱 AI GLM-4-Flash
- **长文本**：月之暗面 Kimi

### Q: API Key 会被上传到服务器吗？

**A:** 不会。API Key 仅存储在本地浏览器，使用 Chrome 的加密存储 API，不会上传到任何服务器。

### Q: 如何更换 AI 服务商？

**A:** 
1. 打开设置页面
2. 选择新的服务商
3. 输入对应的 API Key
4. 点击「测试连接」验证
5. 保存设置

### Q: 可以同时配置多个服务商吗？

**A:** 可以。你可以配置多个服务商的 API Key，随时在设置中切换使用。

## 🤖 AI 功能相关

### Q: AI 生成的内容质量如何？

**A:** 质量取决于：
1. **选择的模型** - GPT-4 > GPT-3.5 > 其他模型
2. **输入的内容** - 输入越详细，输出越准确
3. **提示词优化** - 我们持续优化提示词以提升质量

### Q: AI 生成速度慢怎么办？

**A:** 可能的原因和解决方法：
1. **网络问题** - 检查网络连接，OpenAI 可能需要代理
2. **服务商限流** - 更换其他服务商
3. **模型选择** - 选择更快的模型（如 GPT-3.5-Turbo）

### Q: 提示"API Key 无效"？

**A:** 请检查：
1. API Key 是否完整复制（无多余空格）
2. API Key 是否已激活
3. 账户余额是否充足
4. 尝试重新生成 API Key

### Q: AI 功能收费吗？

**A:** SmartEdit AI 本身完全免费，但 AI 服务商会收取 API 调用费用。不同服务商价格不同，建议选择性价比高的服务商。

### Q: 如何节省 AI 使用成本？

**A:** 几个建议：
1. 选择性价比高的服务商（如 DeepSeek）
2. 使用更便宜的模型（如 GPT-3.5-Turbo）
3. 精简输入内容，避免冗余
4. 合理使用，不要频繁重复生成

## 🎨 样式库相关

### Q: 样式库是否免费？

**A:** 是的，所有样式模板完全免费使用，无需配置任何 API Key。

### Q: 如何添加自定义样式？

**A:** 目前版本暂不支持自定义样式，但这是我们计划中的功能。你可以：
1. 在 GitHub 提交 Issue 建议新样式
2. Fork 项目自行添加样式
3. 等待后续版本更新

### Q: 样式无法插入到编辑器？

**A:** 请检查：
1. 光标是否在可编辑区域
2. 页面是否已完全加载
3. 尝试刷新页面重新加载扩展
4. 查看浏览器控制台是否有错误

### Q: 一键换色不生效？

**A:** 可能的原因：
1. 选中的内容不包含可换色元素
2. 样式被其他 CSS 覆盖
3. 尝试使用「全文换色」功能

## 🖼️ 配图中心相关

### Q: 配图功能必须配置 API Key 吗？

**A:** 不是必须的。不配置也可以使用预设的背景模板。配置后可以搜索 Unsplash 和 Pixabay 的海量图片。

### Q: 图片搜索没有结果？

**A:** 请检查：
1. 是否已配置图片服务 API Key
2. API Key 是否有效
3. 网络连接是否正常
4. 尝试更换搜索关键词

### Q: 图片无法插入？

**A:** 可能的原因：
1. 网络问题导致图片加载失败
2. 图片格式不支持
3. 编辑器权限限制
4. 尝试使用其他图片

### Q: Unsplash 和 Pixabay 有使用限制吗？

**A:** 有的：
- **Unsplash 免费版**：50 次请求/小时
- **Pixabay 免费版**：5000 次请求/小时

对于个人使用完全足够。

## 🛠️ 功能使用相关

### Q: 如何打开侧边栏？

**A:** 有多种方式：
1. 点击扩展图标 → 选择侧边栏
2. 右键菜单 → 智编助手 → 打开侧边栏
3. 点击页面左侧的悬浮按钮

### Q: 侧边栏遮挡了页面内容？

**A:** 
- **左侧侧边栏**：会推移页面内容，不会遮挡
- **右侧智能助手**：使用 Chrome 原生 Side Panel，独立区域，不会遮挡

### Q: 如何关闭侧边栏？

**A:** 多种方式：
1. 点击侧边栏右上角的关闭按钮
2. 按 `ESC` 键
3. 点击页面其他区域

### Q: 划词工具栏不显示？

**A:** 请检查：
1. 是否选中了文字
2. 是否在可编辑区域
3. 扩展是否正常加载
4. 尝试刷新页面

## 🔧 技术相关

### Q: 支持哪些网站？

**A:** SmartEdit AI 可以在任何网页使用，但主要针对微信公众号编辑器优化。

### Q: 会影响页面性能吗？

**A:** 不会。SmartEdit AI 采用按需加载策略，只在打开侧边栏时加载相关功能，对页面性能影响极小。

### Q: 数据会被收集吗？

**A:** 不会。SmartEdit AI：
- ✅ 不收集用户数据
- ✅ 不上传文章内容
- ✅ API Key 仅本地存储
- ✅ 完全开源，代码可审计

### Q: 如何更新扩展？

**A:** 
- **Chrome 应用商店版**：自动更新
- **开发者模式**：
```bash
git pull
npm install
npm run build
```
然后在 `chrome://extensions/` 点击刷新按钮。

## 🐛 问题排查

### Q: 扩展突然不工作了？

**A:** 尝试以下步骤：
1. 刷新页面
2. 重新加载扩展（`chrome://extensions/` → 刷新按钮）
3. 重启浏览器
4. 检查是否有更新
5. 查看浏览器控制台错误信息

### Q: 如何查看错误日志？

**A:** 
1. 按 F12 打开开发者工具
2. 切换到 Console 标签
3. 查看是否有红色错误信息
4. 截图或复制错误信息，在 GitHub 提交 Issue

### Q: 如何报告 Bug？

**A:** 
1. 访问 [GitHub Issues](https://github.com/smartedit-ai/smartedit-ai/issues)
2. 点击「New Issue」
3. 选择「Bug Report」模板
4. 填写详细信息：
   - 问题描述
   - 复现步骤
   - 预期行为
   - 实际行为
   - 截图或错误日志
   - 系统和浏览器版本
