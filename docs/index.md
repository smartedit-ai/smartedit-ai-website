## 为什么选择 SmartEdit AI？

SmartEdit AI 是一款专为**微信公众号创作者**打造的 Chrome 浏览器扩展。它通过侧边栏的形式，为微信公众号编辑器提供丰富的样式模板、AI 智能写作、一键排版和配图搜索等功能，帮助创作者提升内容创作效率和文章质量。

### 🎯 核心优势

- **无缝集成** - 直接嵌入微信公众号编辑器，无需切换页面
- **AI 赋能** - 多家 AI 服务商支持，智能生成高质量内容
- **样式丰富** - 10+ 套精选样式模板，满足各种排版需求
- **操作简单** - 点击即用，无需复杂配置
- **完全免费** - 开源项目，MIT 协议，永久免费使用

### 🚀 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/smartedit-ai/smartedit-ai.git

# 2. 安装依赖
cd smartedit-ai
npm install

# 3. 构建项目
npm run build

# 4. 在 Chrome 中加载扩展
# 访问 chrome://extensions/
# 开启"开发者模式"
# 点击"加载已解压的扩展程序"
# 选择项目的 dist 文件夹
```

### 📦 主要功能

#### 🎨 超级编辑器
- **侧边栏样式库** - 标题、正文、分割线、卡片、列表等样式模板
- **一键换色系统** - 10+ 预设主题色，支持自定义颜色
- **全文一键排版** - 4 套精选模板（简约清新/商务专业/文艺优雅/科技现代）
- **快捷格式工具** - 清除格式、首行缩进、调整行高

#### 🤖 AI 创作引擎
- **AI 标题生成** - 基于文章内容，一键生成 10 个高点击率标题
- **AI 标题评分** - 对标题进行 0-100 分评分，提供优化建议
- **AI 划词改写** - 选中文字即可润色、扩写、缩写、换语气
- **AI 写作助手** - 输入主题，生成文章大纲或完整文章

#### 🖼️ 配图中心
- **Unsplash 集成** - 海量高质量免费图片
- **Pixabay 集成** - 丰富的免版权图片素材
- **一键插入** - 搜索图片后直接插入编辑器

#### 🛠️ 便捷工具
- **左侧侧边栏** - 样式库、AI 写作、配图等核心功能
- **右侧智能助手** - 快捷工具、笔记、AI 助手、页面信息
- **悬浮工具栏** - 页面快捷入口
- **划词工具栏** - 选中文字自动弹出 AI 工具
- **右键菜单** - 快速调用各种功能

### 🎓 适用场景

- ✅ 微信公众号文章创作
- ✅ 内容排版美化
- ✅ AI 辅助写作
- ✅ 图片素材查找

### 💡 技术栈

- **扩展标准**: Chrome Manifest V3
- **前端框架**: React 18 + TypeScript
- **样式方案**: TailwindCSS
- **构建工具**: Vite
- **AI 接口**: OpenAI API / 智谱 AI API / DeepSeek API 等
