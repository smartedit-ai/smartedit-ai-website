### ‚ú® New Features

#### Super Editor

- **Style Library**
  - 10+ curated style templates
  - Title, body text, divider, card, list styles
  - One-click insert into editor
  - Customizable colors and styles

- **One-Click Color Change**
  - 10+ preset theme colors
  - Custom color picker
  - Extract colors from images
  - Apply to entire article or selection

- **Full-Text Formatting**
  - 4 formatting templates (Simple, Business, Literary, Tech)
  - One-click apply to entire article
  - Automatic style optimization
  - Responsive design

- **Quick Format Tools**
  - Clear formatting
  - First-line indent
  - Adjust line height
  - Text alignment

#### AI Creation Engine

- **AI Title Generation**
  - Generate 10 titles from article content
  - Multiple title styles (emotional, suspense, practical, etc.)
  - One-click insert
  - Copy to clipboard

- **AI Title Scoring**
  - 0-100 point scoring system
  - 5 evaluation dimensions
  - Detailed optimization suggestions
  - Comparative analysis

- **AI Text Rewriting**
  - Polish text
  - Expand content
  - Compress text
  - Change tone
  - Selection-based operation

- **AI Writing Assistant**
  - Generate article outlines
  - Create full articles
  - Topic-based generation
  - Customizable parameters

## Version Numbering

We follow [Semantic Versioning](https://semver.org/):

- **Major version** (x.0.0) - Breaking changes or major features
- **Minor version** (0.x.0) - New features, backward compatible
- **Patch version** (0.0.x) - Bug fixes, backward compatible
