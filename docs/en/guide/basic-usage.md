# Basic Usage

Learn the basic operations of SmartEdit AI.

## üéØ Interface Overview

SmartEdit AI provides multiple entry points:

- **Extension Icon** - Quick access in browser toolbar
- **Left Sidebar** - Main feature panel
- **Right Assistant** - Independent side panel
- **Floating Toolbar** - Quick shortcuts on page
- **Context Menu** - Right-click access

---

## üö™ Opening Sidebars

### Left Sidebar

The left sidebar contains core features: Style Library, AI Writing, and Image Center.

**Method 1: Via Extension Icon**
1. Click SmartEdit AI icon in toolbar
2. Click "Left Sidebar" button
3. Sidebar opens on the left side of page

**Method 2: Via Context Menu**
1. Right-click anywhere on page
2. Select "SmartEdit AI" ‚Üí "Open Left Sidebar"
3. Sidebar appears

**Method 3: Via Floating Toolbar**
1. Look for floating button on page
2. Click to open sidebar

**Closing:**
- Click X button in sidebar header
- Click outside sidebar area
- Press ESC key

### Right Assistant

The right assistant is an independent browser side panel.

**Method 1: Via Extension Icon**
1. Click SmartEdit AI icon
2. Click "Right Assistant" button
3. Side panel opens on right

**Method 2: Via Context Menu**
1. Right-click on page
2. Select "SmartEdit AI" ‚Üí "Open Right Assistant"

**Features:**
- Uses Chrome native Side Panel API
- Independent of page content
- Persists across tab switches
- Better performance

**Closing:**
- Press ESC key
- Click close button in panel

## üé® Using Style Library

### Browse Styles

1. Open left sidebar
2. Click "Style Library" tab
3. Scroll to browse available styles
4. Styles are grouped by category:
   - Title Styles
   - Body Text Styles
   - Dividers
   - Card Styles
   - List Styles

### Insert Style

**Method 1: Cursor Position**
1. Click in editor where you want to insert
2. Click style template in sidebar
3. Style inserts at cursor position

**Method 2: Replace Selection**
1. Select text in editor
2. Click style template
3. Selected text applies the style

### Edit Inserted Style

After inserting:
1. Click on the styled element
2. Edit text content directly
3. Modify style attributes if needed
4. Copy and reuse as needed

## ü§ñ Using AI Features

### AI Title Generation

1. Open left sidebar
2. Switch to "AI Writing" tab
3. Paste article content in text box
4. Click "Generate Titles"
5. Wait for AI to process
6. View 10 generated titles
7. Click title to copy or insert

### AI Text Rewriting

1. Select text in editor
2. Text selection toolbar appears
3. Choose rewrite type:
   - Polish
   - Expand
   - Compress
   - Change Tone
4. Wait for AI processing
5. Review result
6. Click "Apply" to replace original text

### AI Writing Assistant

1. Open "AI Writing" tab
2. Enter topic or outline
3. Choose generation mode:
   - Outline Mode
   - Full Text Mode
4. Set parameters (optional)
5. Click "Generate"
6. Review and edit result
7. Insert into editor

## üñºÔ∏è Using Image Center

### Search Images

1. Open left sidebar
2. Switch to "Image Center" tab
3. Enter search keywords
4. Select image source:
   - Unsplash
   - Pixabay
   - Both
5. Browse search results
6. Scroll to load more

### Insert Image

**Method 1: Direct Insert**
1. Position cursor in editor
2. Click image in search results
3. Image inserts automatically

**Method 2: Preview First**
1. Click image to view large preview
2. Check image details
3. Click "Insert" button
4. Image inserts into editor

### Use Background Templates

1. Switch to "Backgrounds" tab
2. Browse gradient or solid backgrounds
3. Click to insert
4. Add text content if needed

## ‚ùì Troubleshooting

### Sidebar Won't Open

**Possible causes:**
- Page not fully loaded
- Browser compatibility
- Extension not enabled

**Solutions:**
1. Refresh page
2. Check browser version
3. Reload extension
4. Try different page

### Features Not Responding

**Check:**
1. API Key configured correctly
2. Network connection stable
3. API quota available
4. Browser console for errors

### Styles Display Incorrectly

**Try:**
1. Refresh page
2. Re-insert style
3. Clear browser cache
4. Use different browser
