## üöÄ How to Use

### Open Image Center

**Method 1: Via Sidebar**
1. Click extension icon
2. Select "Left Sidebar"
3. Switch to "Image Center" tab

**Method 2: Via Context Menu**
1. Right-click on page
2. Select "SmartEdit AI" ‚Üí "Image Center"

### Search Images

**1. Enter Keywords**
- Type keywords in search box
- Supports Chinese and English
- Can use multiple keywords

**2. Select Platform**
- Unsplash - High-quality photography
- Pixabay - Diverse materials
- Or search both simultaneously

**3. Browse Results**
- Waterfall layout displays search results
- Scroll to load more images
- 20-30 images per page

**4. View Details**
- Hover to see image info
- Click image to view large preview
- Check author, dimensions, etc.

### Insert Image

**Method 1: Direct Insert**
1. Position cursor in editor
2. Click image
3. Image auto-inserts

**Method 2: Preview Then Insert**
1. Click image to view large preview
2. Confirm satisfaction then click "Insert"
3. Image inserts into editor

**Method 3: Download Then Use**
1. Click "Download" button
2. Save to local
3. Manually upload to use

### Adjust Image

After insertion:
- Adjust image size
- Add image caption
- Set alignment
- Add link

## ‚öôÔ∏è Configure Image Services

### Configure Unsplash

**1. Get Access Key**

Visit [Unsplash Developers](https://unsplash.com/developers):
1. Register and log in
2. Click "New Application"
3. Fill in application info:
   - Application name: SmartEdit AI
   - Description: Personal use
4. Accept terms of service
5. Create application
6. Copy Access Key

**2. Configure in Extension**

1. Open settings page
2. Switch to "Image Services" tab
3. Paste Access Key in Unsplash section
4. Click "Test Connection"
5. Shows success and ready to use

**Usage Limits:**
- Free tier: 50 requests/hour
- Sufficient for personal use
- Can apply for upgrade if needed

### Configure Pixabay

**1. Get API Key**

Visit [Pixabay API](https://pixabay.com/api/docs/):
1. Register account
2. Log in and go to "API" page
3. Find API Key in personal settings
4. Copy API Key

**2. Configure in Extension**

1. Open settings page
2. Paste API Key in Pixabay section
3. Click "Test Connection"
4. Shows success and ready to use

**Usage Limits:**
- Free tier: 5000 requests/hour
- More than sufficient
- No upgrade needed

### Works Without Configuration

**Even without API Keys, you can still:**
- ‚úÖ Use preset background templates
- ‚úÖ Use gradient backgrounds
- ‚úÖ Use solid color backgrounds
- ‚ùå Cannot search Unsplash images
- ‚ùå Cannot search Pixabay images

## üí° Search Tips

### Keyword Selection

**Use Specific Keywords:**
```
‚úÖ Good keywords:
- "business meeting"
- "coffee shop work"
- "sunset beach"
- "tech background"

‚ùå Bad keywords:
- "image"
- "nice"
- "beautiful"
```

**Use English Keywords:**
- Unsplash mainly has English content
- English keywords yield richer results
- Can use translation tools

**Combine Keywords:**
```
Single keyword: coffee
Combined keywords: coffee shop laptop work

More precise results!
```

### Search Strategies

**1. From Broad to Specific**
```
First try: business
Second try: business meeting
Third try: business meeting office
```

**2. Try Synonyms**
```
If "happy" doesn't find satisfactory results
Try: joyful, cheerful, delighted
```

**3. Use Scene Descriptions**
```
Don't just search: computer
Try: person using computer in office
```

**4. Specify Style**
```
Add style words:
- minimal (minimalist)
- vintage (retro)
- modern (modern)
- colorful (colorful)
```

### Image Selection

**Consider Article Theme:**
- Match image to article topic
- Avoid irrelevant images

**Note Image Quality:**
- Choose high-resolution images
- Avoid blurry or distorted
- Pay attention to composition and color

**Maintain Style Unity:**
- Consistent image style throughout article
- Coordinated color tones
- Uniform size ratios

**Avoid Copyright Issues:**
- Use Unsplash/Pixabay images
- All free for commercial use
- No copyright concerns
