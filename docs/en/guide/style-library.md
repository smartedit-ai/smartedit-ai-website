## üöÄ How to Use

### Open Style Library

**Method 1: Via Sidebar**
1. Click extension icon
2. Select "Left Sidebar"
3. Switch to "Style Library" tab

**Method 2: Via Context Menu**
1. Right-click on page
2. Select "SmartEdit AI" ‚Üí "Open Style Library"

### Browse Styles

1. Scroll through style library
2. Styles displayed by category
3. Each style has preview effect

### Insert Style

**Method 1: Cursor Insert**
1. Position cursor in editor
2. Click style in library
3. Style auto-inserts at cursor position

**Method 2: Replace Selected Content**
1. Select text in editor
2. Click style
3. Selected text applies style

### Edit Style

After insertion:
- Modify text content
- Adjust style details
- Copy and paste to reuse

## üí° Usage Tips

### Style Matching

**Maintain Unity:**
- Use same style set throughout article
- Keep title styles consistent
- Coordinate colors

**Clear Hierarchy:**
- Level 1 headings most prominent
- Level 2 headings secondary
- Body text simple and clear

**Moderate Decoration:**
- Don't overuse styles
- Special styles only for key content
- Keep overall simplicity

### Improve Efficiency

**Bookmark Common Styles:**
- Mark frequently used styles
- Quick access

**Use Shortcuts:**
- Memorize common shortcuts
- Reduce mouse operations

**Batch Processing:**
- Use full-text formatting
- Complete at once

### Avoid Common Mistakes

**‚ùå Too Many Styles:**
- Using too many different styles in one article
- Appears messy

**‚úÖ Simple and Unified:**
- Choose 2-3 main styles
- Maintain visual unity

**‚ùå Chaotic Colors:**
- Using multiple uncoordinated colors
- Affects reading experience

**‚úÖ Harmonious Colors:**
- Use preset theme colors
- Or choose adjacent colors

**‚ùå Over-Decoration:**
- Adding borders and backgrounds to every paragraph
- Overshadows content

**‚úÖ Highlight Key Points:**
- Special styles only for key content
- Keep body text simple
